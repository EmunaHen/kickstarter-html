Build It Yourself: Your Business, Your Software

Source Code

Source code is the fundamental component of a computer program that is created by a programmer, often written in the form of functions, descriptions, definitions, calls, methods and other operational statements. It is designed to be human-readable and formatted in a way that developers and other users can understand.

Compiled binary file

A compiled binary file is a file, often an executable, created by a compiler from source code. It contains machine-readable instructions, not human-readable text, and is designed to be directly executed by a computer's processor. 

With this 2 definations in mine, here is the story.

That year in 2006, I was looking for an accounting software to manage invoices and other documents. As per all businesses, I bought an accounting software from the market. Furthermore, there was funding from the government with recommended suppliers, so I bought one. However, to my surprise, after 500 transactions the software reached the limit. I complained the issue to the supplier and the software was upgraded to an unlimited version. The limit was released, but the worse was the company profile locked to another company. That is to say the software was purchased under another company name and cannot be changed.

This case was reported to the government and not much can be done. Since then I was set on the journey to develop my own software.

After much research, I found out that all the source codes needed are available in the internet. I studied, copied, complied, and managed to develop an inventory online system. Now, with the help from AI, the direct source code are given when I ask. This has speeded up my development and troubleshooting.

For the very first inventory system, it took me about 3 months to learn and develop. Now, for any application, it should take me a few weeks.

Till now, I have developed applications for accounting, marketing, sales, and operation departments. The developments will be ongoing and nonstop. With full control over the source code, I can add any latest technology available in the market. Furthermore, I can also invent any application that needed by my business.

My message to all of you doing business, do not spend on software, do it yourself. It is all available and free from the internet. Most of all, you have full control over it.

Currently, I have the Invoice and Inventory modules ready to offer and I need another year or so to complete the rest of the modules. Therefore your funding is import for me to complete the last mile.

Also, I need users to give suggestions that will influence my developement so that the application will cover more users' needs.
