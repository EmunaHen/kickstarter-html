Story

Hello!

Sympos is a project for an app that helps you both manage and find clubs locally for in person meetups!

This app aims to revive a public social sphere by helping people create connections on their interests that they can actually spend time with.  Made to be compatible with iOS and Android, it allows users to join or create clubs where users do not swipe through individuals but instead find or start groups as a whole with a set interest. It also allows the user to view and find public clubs or meetups in their feed. It will help anyone in trying to start a book club, find friends to go biking and have gear like you do, help you team up with other concert goers, or even just find a group of people that want to try all the restaurants around town with you!

Some of our key features not used in any other web/App meetup service include:

  • Basket tool: allows a club creator to post needed items/prices for the next meetup, which members can assign to themselves
  • map tool: shows pins of the meetup location, and club creators can even add pictures to help identify the meetup
  • Dates tool: allows you to input your available days to meetup then calculates each users answers to determine the best meetup dates
  • RSVP: is a function with a list of upcoming meetups you can either rsvp or notify you’ll be absent. This helps the creator plan better for resources on the meetup

There are more tools than this to make managing easy in your personal profile and club creators have features and abilities the rest of the club members don’t!

As a club creator OR finder, you can set limits by age group and number of ppl allowed in the club to the best of your comfortability.

As part of Kickstarters disclosure policy, the app creator has used GitHub Copilot to edit any errors in code and use the correct element names for functions, as well as regulating the setup of server elements. The app creator also uses Node.js to translate the code originally in JavaScript to React Native compliant code making the app iOS and Android compatible.

The App also welcomes and encourages uses for organizing volunteer work, church, social rights campaigns, or education! Imagine you could find or start a club where you could help teach others your life skills/experience, or come together to discuss current events to learn as a group! The unique setup has the express purpose of helping you find your local community, to love, learn about, or aid.

You can join as many clubs as you want, and even create a few of your own!

By helping with this project, you’d be building a social connection method that fills a hole in our current need for a closer community. Thank you so much for your time, and we hope you are excited about this project as much as we are!
